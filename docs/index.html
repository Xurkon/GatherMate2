Features

📍 Dual Map Display

Nodes appear on both world map and minimap for easy navigation.

🔄 Auto-Collection

Automatically records new node locations as you gather resources.

📤 Data Sharing

Share your node database with party and guild members.

🗺️ Routes Compatible

Works seamlessly with Routes addon for optimized farming paths.

🎯 FarmHud Compatible

Pins display correctly on FarmHud's transparent HUD overlay.

🎨 Customizable Icons

Adjust icon size, alpha, and minimap display range to your preference.

📥 Import/Export

Merge data from GatherMate2_Data or import from other sources.

⚡ Optimized

Efficient memory usage even with large node databases.

Commands

/gathermate2 Open configuration panel
/gm2 Shortcut to open configuration

Installation

  1. Download the latest release
  2. Extract both folders to Interface/AddOns/:
    • GatherMate2
    • GatherMate2_Data
  3. Enable both addons in character select
  4. Configure via /gathermate2 command
  5. Import pre-populated data from the config panel
